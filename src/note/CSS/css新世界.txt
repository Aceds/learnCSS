下面这段代码既可以去除inline-block元素间的空白间隙，又能保持空格特性：
.space-size-zero { font-size: .1px; font-size: -webkit-calc(1px - 1px); }
理论上讲，直接使用font-size:0就可以实现想要的效果，但是在IE浏览器中直接设置font-size:0会失去空格特性，如无法实现
两端对齐效果等，因此只能设置成font-size:.1px，此时字号大小按照0px渲染，空格特性也保留了。但是，这种做法又带来另外一个问
题，由于Chrome浏览器有一个12px的最小字号限制规则，因此font- size:.1px会按照font-size:12px渲染，怎么办呢？我们使用一
个IE浏览器无法识别的语法就可以了，这里就使用了-webkit- calc(1px - 1px)

默认换行行为：
1、中文
所有的中文文字内容都是换行点
如果有标点：逗号，句号，问号，逗号，感叹号就属于避头标点，是不能在开头显示的，称为避头标点;
            上引号，上括号不能出现一行的尾部，称为避尾标点。
    如果希望中文内容不是换行点，使用word-break:keep-all;
    比如表格中 人员姓名（100）
    如果希望中文标点没避头避尾特性，使用line-break:anywhere;
    连续波折号换行，使用word-wrap:break-word;


2、英文和数字
英文的单词和连续的数字默认是不会换行的
    英文连续的数字换行，使用word-break:break-all;

    默认情况下，英文句子都是在空格位置处换行，如果希望这些空格不再是换行点,使用white-space:nowrap;
    英文标点强制换行:overflow-wrap:anywhere;
    我们可以对比下 word-break:break-all 这个声明，就可以知道 overflow-wrap:anywhere 的作用了。

    <p class="p1">i'm a developer</p>
    <p class="p2">i'm a developer</p>

    p {
    width: min-content;
    border: 2px solid deepskyblue;
    }
    .p1 {
    overflow-wrap: anywhere;
    }
    .p2 {
    word-break: break-all;
    }
